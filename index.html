Customer Churn Prediction

Binary classification · Tabular data

Built a model to predict which subscription users are likely to churn in the next 30 days using transactional and behavioral data.

  • Data: ~75k users, 6 months of activity logs
  • Methods: Feature engineering, XGBoost, class imbalance handling, SHAP
  • Result: +18% recall@top10% vs logistic regression baseline

Time Series Demand Forecasting

Forecasting · Time series

Forecasted weekly product demand to improve inventory planning and reduce stockouts.

  • Data: 3 years of sales for 200+ SKUs
  • Methods: Feature-rich regression, Prophet / SARIMA baseline, cross-validation
  • Result: ~12% MAPE reduction vs historical moving average
Code (GitHub) Report (PDF)

Text Classification for Support Tickets

NLP · Multi-class classification

Automated routing of customer support tickets by predicting issue categories from free-text descriptions.

  • Data: 50k anonymized support tickets
  • Methods: TF-IDF, linear models, BERT-based comparison, error analysis
  • Result: Macro F1 +10pts vs rules-based system
